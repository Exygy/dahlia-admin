## To setup:
* Use ruby 2.2.5
* Use node v8.9.1 (npm v5.5.1)
* install yarn (if you have homebrew you can run `brew install yarn`)
* run `yarn install`
* run `bundle install`
* run `./bin/rails webpacker:install`

## To run server:
* `bin/webpack-dev-server --hot`
* `rails s`
